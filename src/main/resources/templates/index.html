Capture form input and send it straight to your inbox.

Demo is an application that will handle HTML5 form input and send it straight to your inbox without the need of any backend server side programming language.

Get started for free

  • Complete Control.

    Demo supports AJAX which allows you to submit forms without refreshing the page while giving you complete freedom with how you handle the response.

    Say goodbye to page refreshes. Demo supports AJAX giving you complete control with how you handle the response without the need of redirecting the url after submission.

  • Easy to Integrate.

    A plug and play solution. Simply add a standard HTML5 form with some additional provided javascript code, and you are ready to go!

    <form action="https://demo/user/id" method="POST"> <input type="email" name="email" placeholder="Email"> <input type="text" name="subject" placeholder="Subject"> <textarea name="message" rows="6" maxlength="300"></textarea> <input type="submit" value="Submit"> </form>

  • Super Fast & Reliable.

    Demo uses the Gmail's API to ensure reliable and fast delivery while reducing the risks of emails going to spam.
